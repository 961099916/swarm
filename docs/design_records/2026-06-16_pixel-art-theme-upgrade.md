# Architecture Decision Record (ADR) - 测试页面与全局弹窗组件复古像素画风格重构

*   **状态**：已批准 (Approved)
*   **日期**：2026-06-16
*   **架构师**：首席全栈架构师 & 极致代码洁癖者

## 1. 上下文与背景
在实现了 Swarm 小程序全局样式像素化以及手写像素 SVG 遮罩图标组件（替代原有的 TDesign 图标字体）后，系统的整体核心页面（如任务大盘、个人中心、RPG 地图）均已顺利步入 16-bit 复古像素画风中。

然而，现存的测试包答题页面（`packageQuiz/pages/test`）、结果报告页面（`packageQuiz/pages/result`）以及全局弹窗组件（`cyber-modal`）仍然硬编码了大量的高饱和度赛博朋克霓虹（Cyber/Neon）渐变发光线条和磨砂毛玻璃风格。为了消除这种强烈的视觉分裂感，使用户在答题和浏览报告时能与 RPG 小游戏关卡维持 100% 一致的“星露谷物语”经典原木羊皮纸画风，我们决定对这几个模块进行彻底的像素化视觉改造。

## 2. 决策与详细方案
1.  **立绘像素化**：废弃原 `test.js` 中通过发光线条绘制的 `AVATAR_SVGS`。设计 16x16 像素的小巫师精灵 SVG 矩阵，生成 `idle`（正常）、`thinking`（思考）、`surprised`（惊讶）和 `happy`（开心）四种表情的 URL 编码 Base64 字符串，实现极具 16-bit 复古情怀的测评立绘效果。
2.  **样式像素化**：
    *   重构 `test.wxss`：移除全息投影台、扫描线和渐变发光。将对话气泡 `question-card-avg` 改造为木牌样式（米黄底、粗褐框）。将进度条、选项卡片 `option-card` 改造为纯直角与像素凹槽质感。
    *   重构 `result.wxss`：重塑 MBTI 维度的数值指示滑槽与滑块，使其呈现纯像素直角风格。重塑社交词云胶囊标签和底部操作按钮。
    *   重构 `cyber-modal.wxss`：将全局弹窗重构为“像素木板盖盒”，按键采用 3D 物理下沉立体按钮。
3.  **兼容与防御设计**：
    *   对 SVG 图片中的 `#` 字符全部进行 `%23` 编码转义，以防低版本 iOS 渲染白屏。
    *   在答题页面卸载生命周期（`onUnload`）中物理清理 WeakMap 绑定的打字机定时器，杜绝内存泄漏隐患。

## 3. 影响与收益
*   **收益**：
    *   视觉体验达到 100% 一致的 Stardew Valley 16-bit 像素感。
    *   消除了赛博发光流光带来的大体积 CSS 代码（大约精简 30% 无用样式）。
    *   页面过渡和交互更具物理按压感。
