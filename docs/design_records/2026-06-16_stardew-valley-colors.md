# Architecture Decision Record (ADR)

## 星露谷物语 (Stardew Valley) UI 经典配色重构决策

- **日期**: 2026-06-16
- **状态**: 已批准 (Approved)
- **关联组件**: 全局像素风格设计体系 (Retro-Pixel Theme)
- **决策人**: Antigravity (首席全栈架构师)

---

### 1. 问题背景与设计目标
用户提出希望界面配色方案能进一步参考并融入《星露谷物语》（Stardew Valley）官方 UI 的经典配色。

我们的设计目标是：在保持极佳文字阅读对比度（遵循 WCAG AA/AAA 对比度规范）的同时，完美复刻星露谷中温馨、复古、略带沙砾质感和原野泥土气息的 16-bit 像素界面感。

---

### 2. 决策与优化细节
我们针对 `frontend/app.wxss` 里的核心配色变量进行如下重塑：

#### 2.1 基础矿区与原野基色调优
- **`--bg-root`**: `#2d1e13` $\rightarrow$ `#24180f`
  - **星露谷典故**: 经典的星露谷矿井深处以及夜色背景的深焦褐色。这可以更好地衬托前台的明亮羊皮纸信笺卡片，强化界面前后的三维立体深度。
- **`--bg-surface`**: `#f5e3ca` $\rightarrow$ `#f7e5cb`
  - **星露谷典故**: 还原星露谷中信件、图鉴与日志框体所使用的高辨识度暖黄色纸张色泽。

#### 2.2 功能性植物与作物色彩调优
- **`--success`**: `#6aaa3c` $\rightarrow$ `#5c9e31`
  - **星露谷典故**: 星露谷夏日牧场里生机勃勃的“像素牧草绿”，带有大自然特有的沉稳，摒弃了刺眼的亮荧光色。
- **`--warning`**: `#f1c40f` $\rightarrow$ `#f7b218`
  - **星露谷典故**: 丰收季标志性的“向日葵暖阳金”，使高亮提示在复古木框中显得十分温馨和谐。
- **`--danger`**: `#e74c3c` $\rightarrow$ `#ba3128`
  - **星露谷典故**: 灌木丛中熟透了的“像素野莓红”，富有沙砾复古感，非常契合像素颗粒感。

#### 2.3 粗矿木质框架与文字调优
- **`--border-default`**: `#4a2c16` $\rightarrow$ `#5c381c`
  - **星露谷典故**: 官方最广为人知的“原木框线粗边褐色”，带给界面坚固的手作质感。
- **`--text`**: `#2b190b` $\rightarrow$ `#331b0b`
  - **星露谷典故**: “深焦木黑褐”，模拟像素浮雕刻字字色，确保卡片内部文字对比度完全溢出。

#### 2.4 亮色主题局部覆盖变量同步调优 (`.theme-light`)
- **`--text`**: `#21140a` $\rightarrow$ `#261408`
- **`--text-secondary`**: `#2d1a0d` $\rightarrow$ `#351d0c`
- **`--text-muted`**: `#4a2c16` $\rightarrow$ `#4c2810`

---

### 3. 架构决定与兼容
- **纯 WXSS 全局样式解耦**: 该重塑 100% 通过 WXSS 变量更新，完全不侵入已有的 DOM 结构，保持了轻量高效的特点。
