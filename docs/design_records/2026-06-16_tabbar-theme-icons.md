# ADR: TabBar 拟态主题图标生成及配置设计决策

## 上下文
微信小程序原生 TabBar 的配置仅支持本地静态图片，且当前平台仅显示了文本，缺少图标。为了提升系统整体的企业级视觉观感与极客气息，决定采用自动化脚本绘制出一套高水准的扁平拟态图标，并完成 pages.json 的原生挂载。

## 决策
1. **统一图标美学**：使用精致、高质感的多色阶线条拟态画风。普通态设为 `#8C8C8C`（中性灰），选中态设为 `#1677FF`（蚂蚁蓝），以高度契合系统的双色主题底座，防止因背景色高频变动而引起的闪烁。
2. **自动化生成脚本**：编写 `generate_tabbar_icons.py`，采用 Python Pillow 库提供的反锯齿弧线渲染，将以下四组图渲染为 `81x81` 像素的透明背景 PNG：
   - 任务 (Task)：看板与终端框。
   - 部署 (Deploy)：云朵与冲天火箭头。
   - 积分 (Credits)：带双环环形芯片/算力晶体。
   - 我的 (Profile)：盾牌与宇航极客面庞线条。
3. **原生配置接入**：更新 `pages.json` 配置中的 `tabBar.list` 以支持这 8 个静态图标文件的动态切换渲染。

## 影响
- 提升了小程序的交互感知。
- 不影响性能，依靠原生系统组件渲染。
